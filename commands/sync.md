# Weave Sync

Sync the Weave workflow across all active AI platforms.

**First**: Check if `.weave/config.json` exists and has a non-empty `generatedAt` field. If not, tell the user: "Weave hasn't been set up yet. Run `/weave:onboarding` first." and stop.

## What to do

1. **Read Weave's source of truth**
   - `.weave/config.json` — project and user config
   - `.weave/agents/` — universal agent definitions
   - `.weave/skills/` — universal skill definitions
   - `.weave/teams/` — team definitions
   - `.weave/rules/` — universal rules

2. **Read active platforms**
   - Read `.weave/platforms.json` for configured platforms

3. **Translate and sync for each active platform**

   **Claude Code (`.claude/`)**
   - Agents → `.claude/agents/*.md` (subagent format with frontmatter)
   - Skills → `.claude/commands/weave:*.md`
   - Rules → `CLAUDE.md`
   - Hooks → `.claude/hooks/`

   **OpenClaw (`skills/weave/`)**
   - Skills → `skills/weave/sub-skills/*/SKILL.md`
   - Agents + Rules → `skills/weave/SKILL.md`
   - Identity → `SOUL.md`, `STYLE.md`
   - Multi-agent → `AGENTS.md`
   - Hooks → `hooks/`

   **Cursor (`.cursor/`)**
   - Commands → `.cursor/commands/weave-*.md`
   - Rules + Agents → `.cursor/rules/weave.mdc` (with `alwaysApply: true` frontmatter)
   - Hooks → `.cursor/hooks.json` + `.cursor/hooks/`

   **Codex (`.agents/`)**
   - Instructions + Agents → `AGENTS.md`
   - Skills → `.agents/skills/weave-*/SKILL.md`
   - Hooks → `.codex/hooks.json` + `.codex/hooks/`

4. **Report what was synced**
   Show which platforms were updated and what changed.

5. **Update state**
   - Update `lastSync` in `.weave/platforms.json`
   - Log the sync in `.weave/history/`

## Important
- Never overwrite platform-specific customizations that weren't created by Weave.
- Weave-generated content is marked with `<!-- Generated by Weave 🧶 -->` headers.
- If a file has both Weave and non-Weave content, only update the Weave sections.
