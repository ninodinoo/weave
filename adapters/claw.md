# Adapter: Claw (Claude Desktop)

## Directory Structure
Claw uses the same `.claude/` directory structure as Claude Code.

```
project/
├── .claude/
│   ├── commands/          # Slash commands (Markdown files)
│   ├── agents/            # Agent definitions (Markdown files)
│   └── hooks/             # JavaScript hooks
├── CLAUDE.md              # Project-level rules
└── src/
```

## Supported Features
- ✓ Slash Commands
- ✓ Agents / Subagents
- ✓ Rules (CLAUDE.md)
- ✓ Hooks
- ✓ MCP Servers

## Translation from Weave Universal Format
Same as Claude Code — identical file structure and format.

## Notes
- Claw (Claude Desktop) shares the `.claude/` config format with Claude Code
- Installation is identical to Claude Code
- All features work the same way
